## Hardest Challenge

**Challenge**: Real-time processing on limited hardware while maintaining 90%+ accuracy.

**Solution**:
I implemented frame skipping and multi-threading to process video efficiently:

```python
import threading
from collections import deque

class VehicleDetector:
    def __init__(self, model_path):
        self.net = cv2.dnn.readNet(model_path)
        self.results_queue = deque(maxlen=10)
    
    def detect_vehicles(self, frame):
        blob = cv2.dnn.blobFromImage(frame, 1/255, (416, 416))
        self.net.setInput(blob)
        outputs = self.net.forward()
        return self.parse_outputs(outputs)
    
    def process_video(self, video_path):
        cap = cv2.VideoCapture(video_path)
        frame_count = 0
        
        while True:
            ret, frame = cap.read()
            if not ret:
                break
            
            # Process every 3rd frame for speed
            if frame_count % 3 == 0:
                detections = self.detect_vehicles(frame)
                self.results_queue.append(detections)
            
            frame_count += 1
```
